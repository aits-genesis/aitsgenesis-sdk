## Installation

> **GitHub Packages** requires authentication. Add the following to your project's `.npmrc`:
>
> ```ini
> @aits-genesis:registry=https://npm.pkg.github.com
> //npm.pkg.github.com/:_authToken=${GITHUB_TOKEN}
> ```

```bash
npm install @aits-genesis/core
```
